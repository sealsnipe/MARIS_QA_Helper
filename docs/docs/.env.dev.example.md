# `.env.dev.example`

**Quellpfad:** `.env.dev.example`

## Zweck

Vorlage für die lokale Entwicklungsumgebung (`.env.dev`). Wird von `scripts/dev_local.sh` und `docker-compose.dev-local.yml` genutzt, wenn keine bestehende `.env.dev` vorhanden ist. Enthält Pfade und Ports für SQLite unter `data-dev/`, Qdrant auf localhost und Dev-freundliche Cookie-Einstellungen.

## Ablauf (kurz)

1. Kopie nach `.env.dev` anlegen (manuell oder automatisch durch `dev_local.sh`).
2. `OPENAI_API_KEY` und ggf. Codex-OAuth-Pfade setzen.
3. `dev_local.sh` sourced `.env.dev` und startet Qdrant sowie Uvicorn mit diesen Werten.

## Konfiguration / Parameter

| Variable | Beispielwert | Beschreibung |
|---|---|---|
| `APP_HOST` | `0.0.0.0` | Bind-Adresse des FastAPI-Servers |
| `APP_PORT` | `8088` | HTTP-Port der Anwendung |
| `SESSION_SECRET` | `dev-local-change-me-…` | Geheimschlüssel für signierte Sessions (Dev-Platzhalter) |
| `DATABASE_URL` | `sqlite:///./data-dev/support_kb.sqlite3` | SQLite-Datei für lokale Dev-Daten |
| `QDRANT_URL` | `http://127.0.0.1:6334` | Qdrant-REST-Endpoint (Dev-Port, oft 6334) |
| `COLLECTION_PREFIX` | `kb_` | Präfix für Qdrant-Collections pro Kunde |
| `OPENAI_API_KEY` | `sk-your-key-here` | OpenAI- oder kompatibler API-Key (Pflicht) |
| `OPENAI_BASE_URL` | `https://api.openai.com/v1` | Basis-URL für Embeddings/API |
| `EMBEDDING_MODEL` | `text-embedding-3-small` | Embedding-Modellname |
| `EMBEDDING_DIM` | `1536` | Vektordimension (muss zum Modell passen) |
| `LLM_AUTH_MODE` | `chatgpt_oauth` | `api_key` oder `chatgpt_oauth` für Chat/Agent |
| `CHAT_MODEL` | `gpt-5.4-mini` | Modell für Agent-Antworten |
| `CODEX_AUTH_PATH` | `~/.codex/auth.json` | Codex-Auth-Datei bei OAuth-Modus |
| `CODEX_OAUTH_AUTH_PATH` | `~/.oauth_codex/auth.json` | Alternative OAuth-Auth-Datei |
| `CODEX_BASE_URL` | `https://chatgpt.com/backend-api/codex` | Codex-Backend-URL |
| `SESSION_COOKIE_SECURE` | `false` | `true` hinter HTTPS (Prod) |
| `DEPLOY_PROFILE` | `dev` | Profil-Hinweis (`dev` / `prod`), von Setup-Skripten gesetzt |
| `TOP_K_DEFAULT` | `4` | Standard-Anzahl Retrieval-Treffer |
| `MIN_SCORE_DEFAULT` | `0.25` | Mindest-Score für Vektorsuche |
| `MAX_TOOL_ROUNDS` | `4` | Max. Agent-Tool-Schleifen |
| `MAX_UPLOAD_MB` | `30` | Upload-Größenlimit in MB |
| `ALLOWED_EXTENSIONS` | `.txt,.md,.pdf,.docx` | Erlaubte Dateiendungen beim Upload |
